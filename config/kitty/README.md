Docs: <https://sw.kovidgoyal.net/kitty/conf/#sample-kitty-conf>

```bash
# install kitten
kitty +kitten themes
# choose theme interactively (very slow)
kitten themes
# set themes by name
kitten themes 'gruvbox dark'
```
